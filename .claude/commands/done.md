# /done - Finaliser un ticket

Merge PR, update ticket, cleanup worktree - tout automatique.

## Workflow

1. Vérifier CI checks (attendre si en cours)
2. Merger la PR (squash + delete branch)
3. Passer ticket Linear en "Done"
4. Supprimer le worktree
5. Pull main

## Argument

Numéro PR ou ticket: `9` ou `PAR-136`
